This dataset contains the scripts and
[StepUp workflow](https://reproducible-reporting.github.io/stepup-core/stable/)
to regenerate the "AutoCorrelation Integral Drill" (ACID) dataset,
as well as the generated time series.

The workflows used for the validation of STACIE, a software package that computes autocorrelation integrals,
along with the corresponding validation results, can be found in the [acid-test Git repository](https://github.com/molmod/acid-test).
